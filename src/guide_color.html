Brand Color palette

해당 브랜드의 기존 Identity Color와 Brand Color Palette을 바탕으로 하되 App의 목적과 App이 사용되는 플랫폼, 디바이스 환경과 사용자층을 고려하여 App의 Color Plaette를 구성한다. Brand Color palette는 일반적으로 Primary Color, Secondary Color, Point Color 로 구성되는 경우가 많지만 필수적인 요건은 아니다. 필수적으로 정의되어야 할 Brand Color는 Primary Color 그리고 App UI 전반에 적용될 Gray Scale 이다.
